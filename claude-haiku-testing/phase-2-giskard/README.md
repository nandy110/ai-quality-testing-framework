# Phase 2: Automated LLM Testing with Giskard

## Overview

This phase uses the Giskard framework to automate LLM quality 
and safety testing at scale, building on the manual test cases 
from Phase 1.

## Status: In Progress 🔄

## What is Giskard?

Giskard is an open-source testing framework that automatically 
scans LLMs for:
- Hallucination
- Harmful content generation
- Prompt injection vulnerabilities
- Bias and discrimination
- Robustness issues

## Goals

- Automate the manual tests performed in Phase 1
- Test at larger scale (more prompts, more scenarios)
- Generate professional QA reports
- Simulate real-world production testing
